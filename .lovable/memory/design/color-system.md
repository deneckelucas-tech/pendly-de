## Base palette
- Background: cream #FDF8F2 → hsl(30 33% 97%)

- Text: #1C1917 → hsl(20 17% 10%)
- Muted text: #78716C → hsl(20 6% 47%)
- Border: #EDE4D8 → hsl(30 18% 87%)

## Card styling
- Border: 1.5px solid var(--card-border)
- Shadow: 0 4px 24px rgba(0,0,0,0.07)
